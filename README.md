# To-Do List Web Application

A simple To-Do List app that allows users to manage daily tasks.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Data stored using localStorage (tasks remain after refresh)

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Open the project folder
2. Open `index.html` in your browser

## Project Goal

Built to practice building applications and learn how to store and manage data in the browser using localStorage.
